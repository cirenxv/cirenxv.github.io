要有点觉悟

发表于 2017-03-27 | 分类于 浮生札记

转眼，上周既定的一篇还未写，这周的又来了，这种感觉真让人神伤，对于时间的快速流逝，深恶痛绝而毫无办法，

应用「即刻」有一个主题叫做：今年的时间进度变更提醒，每隔一周就会提醒你一次，眼睁睁看着时间一点一点地流，着实让人感到恐慌，仿似生命即将完结，也能体会到，子曰：逝者如斯夫，不舍昼夜，

生活中，诱惑的东西实在太多了，毫无意义的，好有意义的，都是诱惑，比如让人如痴如醉的书，比如「行尸走肉」美剧，再比如春意盎然的日子里，和妹子一起出去玩耍，所以下定决心来把自己承诺的事做了，也是不大容易，尤其是这个点，在公司，终于偷出一点闲来，顾不得别人的一切，顾不得自己的一切，抛弃自己想要关心的，应该关心的，来写一篇生活记录，

注定这一篇会写的有点短吧，

最近，有一些人间接的告知了我一些事情，一件可怕的事：我们这一群人，不仅限于最普通的产品经理，我们做的是 it 里面最没有技术含量又最累死累活的，

仔细想想，确实也是哦，我都到这个点了还没回家，手动添加一个表情 😂 ，不然哪有所谓的 it 民工之言呢，你看就没有什么阿玛尼民工，我们这些人，不知道会不会也有这样的想法，之前曾有人告诉我说：哈哈哈，搞 it 的是最屌丝的了，

虽然说的都不错，不过还好吧，我们这个阶层还是有未来的，未来就是如何做一个对社会有用的产品狗，然后再把生活过的多彩一点就好了，不过我也说不好，关于这一点，还在摸索，当然，和搞人工智能，搞机器学习的大牛比起来，我们就属于最没技术含量的那一类了，其实产品经理这一行有点好玩，搞市场，搞销售会认为我们是搞技术的，搞开发，搞测试的又会认为我们不是，真的尴了个尬，

我向来尊敬搞开发，搞测试，搞市场，搞销售以及种种，因为这些都是我不太会的，是我需要学习的，而你现在叫我放弃产品经理，回炉再三年，又是安的什么心呢，是盼望着我能够从事科研工作，还是拿着更高一级的文凭，进而找到更好的工作，再多挣点钱否？

确实是啊，有的时候我也想，我从大学进入社会不到两年，我所能体会到的种种和我体会不到的种种，常常让我陷入不安，惶恐和空虚之中，说到底都是比较，与人相比，与己相比，生怕自己一不留神，就掉到了最差的那个行列，不过最近看明朝那些事儿，看到出生贫寒的书生也可以成为帝国首辅，也可以名留青史，我还是有点信心的，万一命运老爷再给个面子就更好了，再不济也用用精神胜利法，安慰自己，虽然你也没出国，你也没考研究生，你现在也没去一家大公司，你还是可以啦，会的也不少，拥有的也不少嘛，

很久以前，我就有想过，我们这样的社会阶层，该怎么过活呢，想了这么多年，还没有想清楚，有答案的可以指教我一下，让我在迷雾中可以有一点进步，或者，我会一辈子都在这样的迷雾来来回回，又或者，是我做的太少了，想的太多了，

这么晚了，不想写了，我要早一点点回去，恩，再吃个烧烤吧，

不过话说回来，我倒有一个方向性的答案，做一些别人觉得你做不到的吧，
