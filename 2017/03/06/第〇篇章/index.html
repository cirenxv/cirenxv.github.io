第〇篇章

发表于 2017-03-06 | 分类于 浮生札记

拥有个人博客，这个想法已经想了很久了，
大学时期，曾经因为这个想法接触了 WordPress，但后来，因为一些我已记不得的原因，就渐渐放弃了，

个人素来有一些写文字的想法和动力，初中，我常在课上课余，模仿着古人的风韵写一些所谓的诗词，到高中、大学，人生好似走到一个必然要经历的感伤阶段，这个时期的我，常常沉浸在生活、爱情、成长所带来的各种欣喜与痛苦之中，情感也尤为发达，直到现在，我也因这个时期所遇见的人、物、事，打心里蒸腾出一丝五味杂陈，正因如此，这段时光也是我在文字造诣上出产最多的，我常常会感谢那时的我，他遇见了很多影响我一生的人，也做了一些相对疯狂的事，我想着，这些大概称为是一个名词：人生财富，

现在，如今，褪去学生时代青涩的魔幻主义，与我个人，反在趋于规律化的生活中很难找到快乐，或者说个人心灵上的满足感，

这种消极的感受，其实也想了很久，并得出一些并不完整的结论，不过这没关系，我们无需陷入完美主义的逃避怪圈，

其一，我在未知的未来面前表现出了恐惧，和不愿挣脱现有的安全感，
其二，不觉间，我抛弃了很多以前的爱好，这让我深感不安，
其三，作为一个互联网人，我需要面对不断意识到的自身缺陷，和与人并论的不足，以及循环学习的焦虑，

但，这段时间，我慢慢悟出一些自己的道理，其实吧，快乐并不只来源于耗费大量时间和精力做的事，有时我们苦苦追寻的也许并不只在目的地，例如走出门去，和友人吃个饭，再陪女票去逛花市，这些都足以让人感受到满足感，我想说在苦苦追寻的路边，也可能有我们忽略的，

另有一点很重要，我认为个人自封其实很不好，但这并不像说着如此明显，例如我常常会想，“我这个人就是这样的，也没觉得有什么不好”，此乃个人自封，认为人各有各的活法，陷于自我保护和自我满足，当然，想要和别人活得不一样并无错误，只是有时反而让你限制，所以，走出去还是很正确的说法，不论是物理上还是心灵上，结实一些新朋友，走一些从未走过的路，或者换一种走的方式也好，

还有一面，现代化社会，我们被压迫必须从事着一些零零散散的生活和工作，所谓的碎片化，对于个人来讲，我还是喜欢沉浸的感觉，也就是能够长时间的做唯一事，大部分人应该也如此吧，可以说我表现出的对于短的时间的恐惧和不接受已然到了一种病态，我的烦躁和焦虑很大一部分也源于此处，

因此种种，我实在需要找到一种方式，一种自我解脱的方式，最后的结论便回到了一开始，首先，得做一件值得持之以恒做的事，这就是个人博客，也许这个决定并不惊人，这件事本身也很普遍，可我相信她可以给我带来正向的磁场，

再往后的，也有相应的畅想，或许我可以把她经营起来，做一些有意思的分享，或许仰仗生活的流水账认识一些有意思的朋友，或许再来复用于一个公众号？这都是很好的后话，

从今往后看，会要求自己一些必要的事，

  1. 把个人博客坚持下去，每周一篇吧，这一年，
  2. 能捡回自己的爱好，纵使不是大量时间，也可偶尔自娱一下，
  3. 有规律和有张弛的学习，不要过于压迫自己，以致于自惭形秽，
  4. 研究一项未知的，例如深入学习一套编程语言，或者 pmp，提醒不用操之过急
  5. 对未来（不只是职业上）做一个大体的规划，

到了这里，开天辟地的部分就差不多了，原因来由絮絮叨叨也说了很多，我想提一下最近遇到的有趣的人，

一个设计师，男设计师，平时朋友圈里最多的是做女士首饰，且十分华丽，
一个做出了竹制简单车的帅气匠人，主业应该是玩飞盘，
还有咱们部门新来的清新脱俗女产品汪，

不管如何吧，认识新的朋友多少让人有点小确幸，因而产生的各种可能都引人遐想，如同从今开始的个人博客，

我真的发现，功底不如从前了呀，
