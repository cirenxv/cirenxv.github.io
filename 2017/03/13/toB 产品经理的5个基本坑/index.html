toB 产品经理的5个基本坑

发表于 2017-03-13 | 分类于 狗的修行

上周，工作上比较忙碌，生活上比较懒惰，而近来又沉于《银河帝国》和《明朝那些事儿》两部超长篇小说中深深不可自拔，于是乎打整和完善博客的事拖着也没做，心理上略有惭愧，

工作的忙碌体现在不断地外出跑动上，跑动什么呢，跑动的是用户，现在的我如愿成为一只产品狗，且在近一年内所做的都是 toB 的产品，上周不断地拜访客户，实实在在与客户交流，得到客户恩惠，以及产品试运行成功，对于我来说都有些触动，加之近来所看的一篇关于 toB PM 的文章，进而产生了一些思考，现在，把这些想法记录下来，用以分享和自勉，
原文是一篇来自人人都是产品经理的文章，讲了5点 toB PM 易踩的坑，这5点都算是显而易见，却易被忽视，尤其与我来讲需要学习和执行的，
首先摆出结论如下，

  1. 畏惧客户，若不是客户主动联系绝不会主动多问一句
  2. 觉得关注产品研发进度只是项目经理的事
  3. 不和产品的实际使用者确认产品 Demo
  4. 不重视文档更新、部门内的报联商
  5. 不做产品复盘会

1.畏惧客户，若不是客户主动联系绝不会主动多问一句

第一点来讲，其实应该承认我是有这个毛病的，至少还需努力，一般来说大部分客户并不知道自身所想要，也描述不清，靠几次简单的沟通很难到达需求澄清的目的，光靠自身的臆想当然更不可行，我还记得在上一份工作中访谈过一位小学英语老师，现在想起来，真的何其失败，没有足够的准备，而且坦白讲，我也有些羞于和陌生人沟通（有的用户同样如此），于是那次的访谈结果就是毫无所获，
通过和客户接触，我对这一点深有体会，其实客户沟通没有那么困难的，需要的是认真的准备，沟通的勇气，以及主动的心态，当然沟通本身是一门很奇妙的艺术，能不能说服别人，能不能挖掘出用户的潜意识、真需求，都是需要继续修行的，

2.觉得关注产品研发进度只是项目经理的事

现在我们公司也正在实行项目经理和产品经理分管制度，当然这是基于产品经理已忙不过来的情况，对于我们 PM 来说是好事，不过即便产品不再负责项目管理的工作，也是需要密切关注的，之前由我们兼任项管的时候确实做的不好，到了现在的分工下，产品依然需要关注产品研发进度，这是与产品发布，产品运营推广都息息相关的，同时，在大环境下，越来越多的公司要求产品可以兼任这一职责，了解和学习 pmp 也是产品狗成长道路上重要的一步，

3.不和产品的实际使用者确认产品 Demo

让产品实际使用者在设计阶段就参与进来是件不错的事，拿我现在接手的产品来说，通常我们首先沟通的都是企业的管理者，他们其实不关心产品的真正使用状态，只会在意产品功能能否覆盖到心之所想，真正的使用者正是店里的收银员、服务员，尤其是在 PM 对于行业业务的理解还不够深刻，对于用户样本积累数量还不足的情况下，这一点是相当重要，尽早的让真实用户参与进来，让其感受到你是真正为他努力，也许可以获得意想不到的产品改进，甚至是一些你永远不会想到的，在方法论上，原文中所提到的“卡片分类法”、“冒烟测试”都是不错的方法，

4.不重视文档更新、部门内的报联商

这一点也是说来容易，一说就懂，文档更新后当然需要通知所有的项目成员，项目有变化当然也需要让大家信息同步，不过做起来可不简单，在实际操作中，确实存在需求变更后疲于更新需求版本说明，或者在工期短、需求变更频繁的情况下，我们常常都会因为忙碌而选择性忽略这些看似非重要的点，当然，事后产品、开发、测试就要撕逼了，而且报联商也着实花费工作中相当的时间，不然就不会有那么多的协同和沟通软件来解决这些痛点了，如何做到项目成员、部门成员的信息实时对称，实在是一件痛苦却不得不解决的事，除了寻找好的方法和工具，不厌其烦的坚持才是对 PM 的最大考验，

5.不做产品复盘会

产品复盘会目前我们做的很少，通常只在产品遇到重大问题时才做，例如出现大范围影响客户的 bug，或者发布时间严重延期，
在读了这篇文章之后，我才忽然意识到这是个很大的问题，确实，我们不能只关注大的错误，一会导致团队不在乎小的问题，二会使团队过于关注失误，对士气不利，
每一个产品版本都会存在得与失，每一次复盘都应该是对于得的庆贺和对于失的反思，正反两面的结合才能使团队健康的发展，同时这样的复盘会也是 PM 实现自我能力迭代的机会，

说完了，总结一下，其实可以看到，上述5点都是一种沟通过程，与用户沟通，与团队沟通，与自我沟通，同时要做到主动和有效，

最后，周末的时候我和女朋友一起去了一趟奎星楼街，这是一条文艺气息和美食文化浓厚的小巷，街头第一家叫做“成都吃客”的店很值得推荐，烤鱼片、烤五花、烤大排、烤脑花、烤大茄，还有绝情蛙、捞汁花甲等等等等，缺点是人太多，上菜有点慢，服务员照顾不过来（需求痛点啊），街上的其他店家也几乎是人爆满，等我以后吃了再推荐吧，不过要注意的是下午两点所有的店都打烊了，要么早点去，要么下午5点再去，感觉整条街的美食可以吃够好几个月了，看来要赶紧研究一下如何贴图才行，
戛然而止，完，
